Abstract

This is a utility package for Java which implements methods useful in working with byte arrays. This will commonly help people working with network utilities or reading/writing to binary files. A simple class comprised of various tools used in handling of byte arrays. The most used features include multiplexing, demultiplexing, selecting sub-arrays, and concatenating arrays.
